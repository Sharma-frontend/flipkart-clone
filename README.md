# flipkart-clone

🛒 Flipkart Clone

A simple Flipkart clone created using only HTML and CSS. This project replicates the basic UI of Flipkart’s homepage, including the header, navigation bar, product sections, and footer.

🚀 Features

📌 Responsive homepage layout

📌 Navigation bar with categories

📌 Banner section for offers

📌 Product cards with images and details

📌 Footer with links and copyright

🛠️ Tech Stack

HTML5 – Structure of the website

CSS3 – Styling and layout
